STEP 1: Open Terminal/Command Prompt
────────────────────────────────────────────────────────────────────────────
Command:
  cd C:\Users\Arun\Downloads\clauuu\copilot-api

STEP 2: Install Dependencies (First Time Only)
────────────────────────────────────────────────────────────────────────────
Command:
  bun install

OR if you don't have bun:
  npm install

⏳ Wait 1-2 minutes for completion


STEP 3: Authenticate (First Time Only)
────────────────────────────────────────────────────────────────────────────
Command:
  npm start -- auth

What happens:
  ✓ Browser opens automatically
  ✓ Click "Authorize" on GitHub
  ✓ Tokens are saved locally
  ✓ You're authenticated! ✅


STEP 4: Start the Server
────────────────────────────────────────────────────────────────────────────
Command (Default Port 4141):
  npm start

Command (Custom Port - Example 3000):
  npm start -- -p 3000

Command (With Logging):
  npm start -- -v

✅ Server is now running!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
